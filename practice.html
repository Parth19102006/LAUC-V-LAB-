Practice Problems

To master the techniques of integration, it's essential to practice a variety of problems. Here are some practice problems categorized by the methods of integration:

1. Substitution Method

  • ∫(2x)(x^2 + 1)^5 dx
    • A. (x^2 + 1)^6/6 + C
    B. (x^2 + 1)^5 + C
    C. (x^2 + 1)^6/5 + C
    D. (x^2 + 1)^4 + C



  • ∫sin(3x) cos(3x) dx
    • A. -cos^2(3x)/6 + C
    B. sin^2(3x)/6 + C
    C. -sin^2(3x)/3 + C
    D. cos^2(3x)/3 + C



2. Integration by Parts

  • ∫x e^x dx
    • A. x e^x - e^x + C
    B. e^x + C
    C. x e^x + C
    D. x e^x + e^x + C



  • ∫ln(x) dx
    • A. x ln(x) - x + C
    B. x ln(x) + x + C
    C. ln(x) + C
    D. x + C



3. Partial Fraction Decomposition

  • ∫(3x + 5)/(x^2 - x - 6) dx
    • A. (1/2)ln|x - 3| + (5/2)ln|x + 2| + C
    B. ln|x - 3| + ln|x + 2| + C
    C. (3/2)ln|x - 3| + (5/2)ln|x + 2| + C
    D. (1/3)ln|x - 3| + (2/3)ln|x + 2| + C



  • ∫(2x^2 + 3x + 1)/(x^3 - x) dx
    • A. ln|x| + ln|x - 1| + ln|x + 1| + C
    B. (1/2)ln|x| + (1/2)ln|x - 1| + (1/2)ln|x + 1| + C
    C. 2ln|x| + 3ln|x - 1| + ln|x + 1| + C
    D. (1/3)ln|x| + (1/3)ln|x - 1| + (1/3)ln|x + 1| + C



4. Trigonometric Integrals

  • ∫sin^2(x) cos(x) dx
    • A. (1/3)sin^3(x) + C
    B. (1/2)sin^2(x) + C
    C. -(1/3)sin^3(x) + C
    D. -(1/2)sin^2(x) + C



  • ∫sec^3(x) dx
    • A. (1/2)sec(x) tan(x) + (1/2)ln|sec(x) + tan(x)| + C
    B. (1/2)sec(x) tan(x) - (1/2)ln|sec(x) + tan(x)| + C
    C. sec(x) tan(x) + ln|sec(x) + tan(x)| + C
    D. sec(x) tan(x) - ln|sec(x) + tan(x)| + C



5. Numerical Integration

  • Approximate ∫_0^1 e^(-x^2) dx using the Trapezoidal Rule with n=4.
    • A. 0.7468
    B. 0.7475
    C. 0.7480
    D. 0.7490



  • Approximate ∫_0^π sin(x) dx using Simpson's Rule with n=6.
    • A. 2.0000
    B. 1.9995
    C. 2.0005
    D. 2.0010
